# 映映小助手

映映小助手是一个个人学习项目，旨在帮助剪映用户更高效地处理本地草稿文件。本工具提供如草稿格式转换、素材提取等实用功能，**仅供个人学习与研究使用**。

📚 [插件开源项目](https://github.com/supertools2023/jianying_assistant_plugins)  
📖 [软件下载网站](https://www.supertools.store/)

---

### 核心功能

- **草稿格式转换**：支持对剪映草稿降级功能。
- **草稿批量管理**：支持批量备份、删除草稿，快速处理多个项目。
- **Python 插件扩展**：提供 Python 接口，方便开发者读写剪映加密草稿文件。
