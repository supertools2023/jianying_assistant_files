# 🎬 映映小助手

**剪映用户的全能桌面工具** — 让视频编辑工作流更高效、更简单。

映映小助手是一款专为剪映用户打造的跨平台桌面应用，提供草稿管理、素材处理、资源获取、AI 辅助等一站式功能，帮助您从繁杂的文件管理中解放出来，专注于创作本身。

---

## ✨ 核心亮点

### 📁 草稿管理

强大的剪映草稿管理引擎，让您的项目井井有条。

- **多标签页浏览** — 同时打开多个目录，像浏览器一样自由切换
- **智能搜索与筛选** — 按名称、标签、日期快速定位草稿
- **多种视图模式** — 网格、列表、瀑布流，适合不同浏览习惯
- **标签分类系统** — 为草稿添加自定义标签，告别混乱
- **批量操作** — 批量删除、备份、打包、导出，一键搞定
- **实时监控** — 自动检测草稿目录变化，列表实时更新
- **草稿锁定检测** — 智能识别正在编辑的草稿，避免冲突

### 🎨 预设管理

剪映特效、转场、文本模板的统一管理中心。

- **自动扫描** — 自动发现剪映安装的预设文件
- **分类浏览** — 按类型、标签组织预设资源
- **快捷操作** — 快速预览、删除、导出预设

### 📦 草稿打包与分享

将草稿打包为可分享的安装包，一键导入。

- **`.draftpkg` 格式** — 自有打包格式，完整保留草稿结构
- **双击安装** — 关联文件格式，双击即可安装草稿
- **多包批量打包** — 向导式操作，一次打包多个草稿
- **统一导入** — 支持 ZIP、文件夹等多种格式自动识别导入

### 🔧 素材处理

解决素材管理中的各类痛点。

- **素材路径修复** — 迁移或重命名后素材离线？自动扫描并重新链接
- **批量素材替换** — 用自己的素材替换模板素材，快速批量生成视频
- **重复草稿检测** — 智能扫描多个目录，精准定位并清理重复草稿，释放磁盘空间
- **内置备份恢复** — 利用剪映自带的备份机制，一键恢复误操作的草稿

### 🌐 局域网传输

多台电脑间零门槛迁移草稿和预设。

- **自动发现设备** — UDP 广播自动发现局域网内的其他设备
- **高速传输** — 基于 HTTP 协议，充分利用局域网带宽
- **完整/精简模式** — 可选择包含素材或仅传输草稿结构
- **传输进度跟踪** — 实时显示传输状态和进度

### 🤖 剪映自动化

通过自动化工具操控剪映，提升工作效率（仅 Windows）。

- **一键打开草稿** — 直接从映映小助手打开草稿到剪映
- **批量导出视频** — 自动化导出流程，批量处理多个草稿
- **免打扰模式** — 防止剪映在导出过程中抢夺焦点

### 🧠 AI 智能助手

集成多家 AI 服务，为视频创作注入智能能力。

- **多提供商支持** — 预置 DeepSeek、通义千问、MiniMax、SiliconFlow 等
- **AI 智能断句** — 利用大模型进行字幕智能断句，提升字幕质量
- **插件 AI 能力** — 为第三方插件提供统一的 AI 接口

### 📚 资源中心

一站式发现和获取剪映相关资源。

- **丰富资源类型** — 草稿包、预设、视频、音频、图片、课程、LUT 等
- **多网盘支持** — 百度网盘、夸克网盘、阿里云盘、蓝奏云、123 云盘、腾讯文档
- **分类筛选** — 按类型快速找到所需资源

### 🧩 插件系统

开放的 Python 插件生态，无限扩展可能。

- **Python 运行时** — 内置 Python 环境，插件开箱即用
- **丰富 API** — 提供草稿操作、剪映交互、UI 扩展、AI 接口等插件 API
- **插件市场** — 安装、卸载、管理插件，一键操作
- **开发者友好** — 完善的插件开发文档和调试支持

### ⌨️ 快捷键系统

- **全局快捷键** — 导航、编辑、搜索、视图等全方位快捷键覆盖
- **自定义绑定** — 根据个人习惯自由调整快捷键
- **冲突检测** — 自动检测快捷键冲突，提供解决方案

## 🎯 设计理念

- **macOS 风格 UI** — 精致的磨砂玻璃效果与流畅动画
- **深色 / 浅色主题** — 跟随系统或手动切换，保护眼睛
- **响应式侧边栏** — 可展开/收起的导航栏，灵活利用屏幕空间
- **原生体验** — 自定义标题栏、原生窗口管理，融入桌面环境
